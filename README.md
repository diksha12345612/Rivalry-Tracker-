# Rivalry Tracker

Rivalry Tracker is a MERN-based web app concept for tracking friendly gaming rivalries. The project documentation defines the product scope, required features, reference assets, build instructions, and evaluation rubrics.

## Project Overview

The app lets users create rivalries, add games, log match results, view scoreboards, track win streaks, review match history, and see basic profile statistics.

## Included Files

- `overview.txt` - Project overview and target users
- `scope_of_work.txt` - Scope, features, deliverables, and exclusions
- `assets.pdf` - Branding, colors, typography, icon guidance, and asset notes
- `reference_image.png` - Main reference image in PNG format
- `instructions.md` - Build instructions and technical requirements
- `rubrics.txt` - Evaluation rubrics for the final submission
- `reference_image/` - Additional reference mockups

## Tech Stack

- MongoDB
- Express.js
- React
- Node.js

## Core Features

- User registration and login
- Rivalry creation and invite flow
- Game management inside each rivalry
- Match result logging
- Scoreboard and per-game win tracking
- Current win streak tracking
- Match history view
- User profile stats
