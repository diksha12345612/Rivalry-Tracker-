The project includes the development of a web-based Rivalry Tracker application that lets small groups of friends, siblings, or roommates track ongoing head-to-head gaming rivalries across multiple games. Core features include user registration and login, creation and joining of rivalry groups, adding and managing games within a rivalry, logging match results with winner and date, and viewing an automatically calculated scorecard showing total wins, per-game breakdowns, current win streaks, and head-to-head history.

The application must be built using the MERN stack (MongoDB, Express.js, React, and Node.js) and delivered as a responsive web app covering the following pages: landing/login/register, dashboard, create rivalry, rivalry detail with scoreboard and match history, and a basic user profile. The completed source code (frontend and backend) must be delivered via a public GitHub repository, along with a deployed live demo link.

The work is limited to the MVP feature set described above - authentication, rivalry/game/match management, and scorecard logic - with no real-time notifications, social media login, mobile app development, payment integration, image/avatar uploads, or admin dashboard required.
